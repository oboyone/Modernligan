This page is maintained by Alexander Johansson with support from a number of persons within the Stockholm Modern Community. The aim for the project is to create a high quality statistics database for Modernligan.

The data is sourced from Wizards of the Coasts Eventlink, and scraped using BeautifulSoup. Only events from the 11th of March 2024 are included in the ELO and stats calculations, as earlier events are not parsable from Eventlink

You can find the project on GitHub: https://github.com/oboyone/Modernligan if you are interested or if you want to contribute

The webbapp is written in Python and uses Flask with the bootstrap CSS framework. If you have any questions, please contact the maintainer at: {{ maintainer_email }}
